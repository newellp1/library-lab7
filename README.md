# Library Lab 7

This is a Blazor Server web app for managing books and users.

## Features
- Add/edit/delete books and users
- Borrow and return books
- CSV file persistence
- Unit testing using MSTest
- CI pipeline using GitHub Actions

## How to Run

```bash
dotnet restore
dotnet build
dotnet run
